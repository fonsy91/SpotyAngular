Componente Json

Usaremos la pagina web https://jsonplaceholder.typicode.com/ para poder hacer esta prueba con Json. Para ello lo primero que debemos hacer es importar el modulo de HttpClient en el modulo principal app.module.ts. Hemos hecho que el componente sea un servicio en el apartado de app.componet.ts hemos declarado un constructor y dentro lo siguiente this._http.get('https://jsonplaceholder.typicode.com/users').subscribe(data=> console.log(data)); Con esto y atraves de la URL del Json con los datos y el get obtenemos los datos y nos suscribimos para despues mostrar por consola todo el Json. El modulo o paquete HttpClient se usa para realizar solicitudes a traves de metodos algunos de ellos son: request(), delete(), get(), head8), jsonp(), option(), patch(), post(), put(). Los cuales ya veremos mas adelante. Con suscribe lo que haces es que te den los datos consta de tres parametros el de los datos, el de error y el de finalizacion.
