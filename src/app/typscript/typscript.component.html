TYPSCRIPT

En este apartado veremos una introduccion a typscript para poder utilizarlo luego en angular. TypeScript es un superconjunto escrito de JavaScript, lo que significa que todo el código JavaScript es un código TypeScript válido. Script agrega muchas características nuevas además de eso. TypeScript hace que JavaScript sea más parecido a un lenguaje fuertemente orientado a objetos, similar a C # y Java.
